# HunterBot - Крипто-охотник для безопасных P2P сделок

Telegram бот для автоматического поиска выгодных предложений по покупке/продаже криптовалюты (TON, USDT) на Avito и Юле с системой безопасных сделок через эскроу.

## Функции

✅ **Парсинг объявлений** - автоматический мониторинг Avito и Юлы каждые 3 минуты
✅ **Проверка курса** - сравнение с реальным рыночным курсом TON
✅ **Защита от мошенничества** - проверка продавцов, AI-анализ объявлений
✅ **Безопасные сделки** - эскроу через YooMoney и TON
✅ **Уведомления** - мгновенные оповещения о выгодных предложениях
✅ **Премиум подписка** - без комиссии и приоритетные уведомления

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Заполните переменные окружения в `.env`:
- `BOT_TOKEN` - токен вашего Telegram бота от @BotFather
- `DB_URL` - URL подключения к PostgreSQL
- `YOOMONEY_TOKEN` - токен YooMoney API
- `YOOMONEY_WALLET` - номер кошелька YooMoney
- `MNEMONIC` - мнемоник фраза для TON кошелька
- `ADMIN_ID` - ваш Telegram ID

4. Создайте базу данных PostgreSQL:
```bash
createdb hunterbot
```

5. Запустите бота:
```bash
python main.py
```

## Технологии

- **aiogram 3.x** - фреймворк для Telegram ботов
- **SQLAlchemy** - ORM для работы с БД
- **PostgreSQL** - база данных
- **pytonlib** - работа с TON blockchain
- **YooMoney API** - прием рублевых платежей
- **APScheduler** - планировщик задач для парсинга

## Монетизация

1. **Комиссия 1.9%** с каждой сделки через эскроу
2. **Премиум подписка** 299₽/мес - без комиссии + приоритет
3. **Реклама P2P бирж** - партнерские программы
4. **Белая метка** - продажа решения крупным каналам
